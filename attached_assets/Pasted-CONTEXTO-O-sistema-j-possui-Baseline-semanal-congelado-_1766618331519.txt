# 1) CONFIRMAÇÃO / ALTERAÇÕES

## 1.1 Página “Confirmação e Alterações”
Menu: Operação → Confirmação / Alterações

Funcionalidades:
- Selecionar:
  - setor
  - semana
  - dia específico
- Visualizar:
  - escala planejada
  - convocações emitidas
  - aceites / recusas
  - substituições
- Ações permitidas:
  - cancelar convocação (com motivo)
  - gerar substituição
  - ajustar horário (somente via Adjustment Run)
- Bloqueios:
  - nunca alterar Baseline
  - sempre gerar Adjustment Run com motivo

---

# 2) SUGESTÕES DIÁRIAS (COPILOTO)

## 2.1 Conceito
Sugestões Diárias NÃO executam ações.
Elas apenas recomendam ajustes com base em dados novos.

## 2.2 Fontes de dados
- Novos HPs (ocupação mudou)
- Checkins / Checkouts do dia
- Recusas de convocação
- Eventos de calendário
- Desvio histórico projeção x real

## 2.3 Tipos de sugestões
- “Sugestão de reforço de equipe”
- “Sugestão de redução de horas”
- “Sugestão de antecipar/adiar turnos”
- “Sugestão de substituição preventiva”

Cada sugestão deve conter:
- setor
- dia
- impacto estimado (horas / pessoas)
- justificativa clara
- botão:
  - “Aplicar” → gera Adjustment Run
  - “Ignorar” → registra decisão
